🚀 Pipeline Hub

A unified CI/CD command center for GitHub Actions workflows

📦 Repository Management

📋Multi-repo dashboard — View all org repositories with language badges, visibility status, and default branch info.
🔍Search & filter — Instantly search repos and workflows by name or language.
🌿Smart branch selection — Priority branches (main, dev, uat, prod, RELEASE_*) always listed. Only recent branches (last 3 weeks) shown for large repos.

⚡ Workflow Operations

▶️Run workflows — Trigger workflow_dispatch runs with branch selection and custom input parameters.
🔄Re-run jobs — Re-run all jobs or just failed jobs with a confirmation dialog.
📄Log viewer — View job logs inline without leaving the dashboard.
🔽Job breakdown — Expand any run to see individual jobs, steps, durations, and pass/fail status.

📊 Analytics & Monitoring

📈Failure analytics — Success rates, MTTR, and flaky workflow detection per repository.
🌍Environment matrix — Bird's-eye view of deploy status across dev, uat, staging, preprod, and prod.
⚡Activity feed — Live feed of recent runs across all repos with status filters.
🔔Notifications — Browser notifications when triggered workflows complete. Bell icon shows watched count.

🎨 Interface

🌙Dark / Light theme — Toggle between dark and light modes. Your preference is saved.
🔁Auto-refresh — Dashboard refreshes every 30 seconds. Data is cached for 5 minutes to reduce API load.
